# 贪吃蛇 Snake Game

一个使用 Python 内置 `tkinter` 库编写的经典贪吃蛇小游戏。

## 运行方式

```bash
python3 game.py
```

无需安装任何第三方依赖。

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 控制蛇的移动方向 |
| `W` `A` `S` `D` | 控制蛇的移动方向 |
| `Space`（空格） | 暂停 / 继续 |
| `R` | 游戏结束后重新开始 |

## 游戏规则

- 控制蛇吃到红色食物，每吃一个得分 +10
- 蛇身会不断变长
- 撞到墙壁或自己的身体则游戏结束
- 吃到食物后蛇的移动速度会略微加快
