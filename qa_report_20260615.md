## 4. 已知问题 / 待优化

1. **后端超时**：第一次后台启动的 uvicorn 因默认 600s 超时被 kill，已改为 `disable_timeout=true` 运行。
2. **设置面板关闭偶发不灵敏**：在同时存在全局设置面板和移动端资源面板 DOM 时，通过 JS 选择器关闭偶尔会点到错误元素；直接点击可见 X 按钮可正常关闭。
3. **Live2D 点击互动**：本次仅验证了模型渲染与“鼠标跟踪中”状态，具体的点击触发的表情/动作需要后续更细粒度的交互测试。
4. **移动端 iframe 连接状态**：仅布局验证用途，真实设备无需担心。
5. **输入框残留**：使用 `execCommand('insertText')` 填充后，如果未发送，内容会留在输入框中，属于测试脚本行为，不影响真实用户输入。
